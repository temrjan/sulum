```json

  "source": "authors/01-frankl.md",
  "author": "Frankl",
  "chapter": "4-BOB: TEXNIKA — PARADOKSAL INTENTSIYA",
  "type": "technique",
  "keywords": ["paradoksal intentsiya", "xavotir", "qo'rquv", "hazil"],
  "use_cases": ["xavotir", "fobiyalar", "obsessiyalar"]
}
```

**Kontent turlari (type):**
- `biography` — muallif tarjimai holi
- `concept` — nazariy tushuncha
- `technique` — aniq texnika
- `dialogue` — muloqot namunasi
- `quote` — iqtibos
- `case` — keys/hikoya
- `comparison` — boshqalar bilan taqqoslash
- `indication` — ko'rsatmalar/qachon qo'llash
